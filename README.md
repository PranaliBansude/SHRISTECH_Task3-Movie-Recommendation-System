# Task 3 – AI-Based Movie Recommendation System

## Objective
To recommend movies based on user preferences using AI techniques.

## Description
This system uses a content-based recommendation approach.  
TF-IDF is applied to movie features, and cosine similarity is used to find similar movies.

## Output
The system recommends a list of movies related to the user’s input.

- Python
- Machine Learning
- Recommendation Systems
